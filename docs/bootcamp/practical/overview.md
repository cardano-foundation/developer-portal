# Build

Get your hands dirty. Ship something. Hands-on exercises scaling from your first transaction to multi-validator production systems.

## Stages

| # | Stage | Description |
|---|-------|-------------|
| 06 | Hello World | Set up Mesh SDK, connect a wallet, send your first transaction |
| 07 | Multisig | Multi-signature transactions with a NextJS frontend |
| 08 | Your First Aiken Contract | Write, compile, and deploy a smart contract |
| 09 | Testing Contracts | Test Aiken contracts with mock transactions |
| 10 | Validation Best Practices | Write lean validators, less on-chain work |
| 11 | Reading Blueprints | CIP-57 blueprints, bridging on-chain and off-chain |
| 12 | Vesting Contract | Lockup contract, real DeFi primitive |
| 13 | Multi-Validator NFT | Multi-validator system with state threads |
