In this session of the Cardano Developer Office Hours, Michiel Bellen, Global Head of Integrations at the Cardano Foundation, presents Heron, a new API designed to simplify interaction with the Cardano blockchain. He explains how Heron enables developers to access FastAPI endpoints for wallets, transactions, and policies, and demonstrates how to deploy it using Docker with minimal setup requirements.
